# Blue Horizon Travel

Uma página web moderna e interativa para uma agência de viagens premium especializada no sul do Brasil. O site apresenta destinos turísticos com informações detalhadas sobre pontos turísticos, restaurantes e hotéis, além de uma seção de experiências personalizadas.

## 🚀 Funcionalidades

### 🏖️ Destinos Turísticos
- **9 destinos** no sul do Brasil: Bombinhas, Porto Alegre, Gramado, Balneário Camboriú, Paranaguá, Curitiba, Florianópolis, Foz do Iguaçu e Bento Gonçalves
- **Cards interativos** com imagens de fundo e descrições atrativas
- **Informações detalhadas** ao clicar em cada destino:
  - Pontos turísticos principais
  - Restaurantes recomendados (com telefone, endereço, menu e preços)
  - Hotéis sugeridos (com contato e localização)

### 🎯 Seção de Destaques (Experiências)
- **3 categorias de viagem**:
  - **Retiro à beira-mar**: Bombinhas, Balneário Camboriú, Florianópolis
  - **Vibração urbana**: Porto Alegre, Curitiba, Gramado
  - **Refúgio natural**: Bento Gonçalves, Paranaguá, Foz do Iguaçu
- **Cards clicáveis** que mostram destinos recomendados para cada estilo
- **Toggle de abertura/fechamento**: clique novamente para fechar
- **Imagens de fundo** nos cards de destino com sobreposição para legibilidade

### 🎨 Design e UX
- **Tema escuro luxuoso** com gradientes azuis e roxos
- **Animações de scroll** suaves com Intersection Observer
- **Navegação responsiva** com links para seções
- **Layout responsivo** otimizado para desktop e mobile
- **Botão de fechar** nos painéis de informações

### 📱 Interatividade
- **Navegação suave** entre seções
- **Formulário de contato** funcional com validação
- **Scroll automático** ao clicar em destinos (opcional)
- **Estados visuais** para hover e foco

## 🛠️ Tecnologias Utilizadas

- **HTML5**: Estrutura semântica e acessível
- **CSS3**: Design moderno com variáveis CSS, flexbox/grid, animações
- **JavaScript (Vanilla)**: Interatividade, manipulação DOM, eventos
- **Imagens**: Galeria de destinos do sul brasileiro

## 🚀 Como Executar

1. **Clone/baixe** os arquivos do projeto
2. **Abra** o arquivo `index.html` em qualquer navegador moderno
3. **Navegue** pelas seções e interaja com os elementos

Não há dependências externas - o projeto roda 100% no navegador.
